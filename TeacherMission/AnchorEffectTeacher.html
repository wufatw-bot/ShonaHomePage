教育心理學 × 補教現場

當第一句話，
就決定了學生的未來？

淺談「錨定效應（Anchoring Effect）」在補教現場的影響

「你其實很聰明，只是以前沒有人教對方法。」

「這一班程度很強，你們是今年最有機會考上的學生。」

「這題其實很簡單，去年國中生都會寫。」

你有沒有發現，補習班老師的一句話，常常會悄悄改變學生的情緒、信心，甚至影響後續整學期的學習表現？這背後，其實藏著一個心理學上非常有名的概念——

🧠什麼是「錨定效應（Anchoring Effect）」？

錨定效應，是指人們在做判斷時，會不自覺受到「最先出現的資訊」影響，之後的思考與決策，也會圍繞著這個初始印象進行調整。

⚓

就像船下錨後，即使海流改變，
船仍然會圍繞著那個位置漂動。

第一個數字、第一個評價、第一句話，
常常比我們以為的更有力量。

而在補教現場，老師其實每天都在「下錨」。

✦
情境一

📚 一句「你很有天分」，學生開始相信自己

阿哲原本數學成績一直在40分上下。他其實不笨，只是每次寫錯題目後，就開始慌亂。第一次進補習班時，他很緊張地坐在最後一排。

老師改完考卷後，沒有先責備，而是說：

老師的話 「你其實觀念不差，錯的是粗心。這種學生通常進步最快。」

這句話，成了阿哲新的「心理錨點」。之後每次考不好時，他不再認為自己是「數學白癡」，而是重新看待自己——

  • 我只是粗心
  • 我其實有能力
  • 我還可以再進步

半年後，他從 40 分進步到 78 分。

🔍 詳細解析

老師當下做的，不只是鼓勵，而是重新替學生建立自我認知、能力定位，以及對未來的期待。學生開始用「我是有潛力的人」來看待自己。這種現象，也與教育心理學中的「畢馬龍效應（Pygmalion Effect）」有關：

畢馬龍效應 當老師真心相信學生會進步時，
學生往往真的比較容易進步。
✦
情境二

📚 一句「這很簡單」，反而讓學生崩潰

另一位學生小芸，英文底子較弱。某次老師在台上說：

老師的話 「這題很簡單，國二程度而已。」

結果小芸完全不會。那一瞬間，她心中產生的不是「我要努力」，而是：

  • 別人都會，只有我不會
  • 我是不是太笨？
  • 我不適合念英文

接下來整堂課，她幾乎沒再專心。

🔍 詳細解析

很多老師以為「說簡單」是為了降低學生焦慮，但對基礎弱的學生而言，這句話可能形成另一種錨定：

「不會這題的人，就是差。」

學生之後的學習，就會開始被羞愧感綁架。這也是為什麼有些學生不是不努力，而是從一開始，就被貼上「自己不夠好」的標籤。

✦
情境三

📚 價格與課程包裝的錨定

補習班其實也很常利用錨定效應做行銷：

原價
$120,000
→
限時優惠
$48,000
省下 $72,000

即使學生家長原本根本沒想過「12萬」這個價格，但當「12萬」先出現後，4萬8 看起來就「合理」很多。

🔍 詳細解析

這是經典的「價格錨定」。人腦不太擅長真正判斷價值，反而常透過「比較」來感覺便宜或昂貴。

先看到高價
→
中價感覺划算
先看到難題
→
普通題感覺容易
先被稱讚
→
挫折較能承受
先被否定
→
後面很難建立信心

教育，其實也是心理學。

✦
情境四

📚 排名公布的心理影響

有些補習班很喜歡在牆上貼榜單：第一名、台大醫科、滿級分學生……這其實也是一種錨定。

新生一進門，就會產生：
「這裡是高手聚集的地方。」

這種環境有時能激勵學生，但也可能讓部分孩子產生巨大壓力，尤其是中後段學生，容易覺得自己不屬於這裡，或永遠追不上。

✦
情境五

📚 老師對班級的「第一印象」

正向錨定 「你們這屆看起來很認真。」
→ 學生會開始更傾向表現認真
負向錨定 「這班感覺很難帶。」
→ 學生也可能慢慢朝那個方向發展

因為學生會不自覺符合老師給的「角色設定」。

老師的期待，
常常會變成學生的現實。
✦

🧠為什麼補教現場特別容易出現錨定效應？

因為補習班通常具有幾個特性，形成了錨定效應的溫床：

  • 高壓競爭環境
  • 快速評價與分類
  • 大量考試排名
  • 強調成果與分數
  • 學生對老師高度信任

在這種環境下，老師的一句話，可能比家長講一百句還有影響力。

✦

🌱真正厲害的老師，不只是會教題目

有些老師最厲害的地方，不是解題速度，而是：他能替學生重新定義自己。

舊的自我認知
「我不行」
「我很笨」
「我永遠追不上」
→
新的錨點
「我還沒學會」
「我只是缺方法」
「我還有機會」

而這些轉變，往往都始於一句話。

✦

📌結語

錨定效應告訴我們：人不是完全理性的。很多時候，我們會被最早聽到的評價、數字、標籤，深深影響。

在補教現場裡，老師不只是知識傳遞者，也可能是學生「自我認知」的塑造者。

所以有時候：一句話，可以讓學生開始努力；一句話，也可能讓學生默默放棄。

教育真正重要的，從來不只是教會學生答案，而是讓學生開始相信——

「自己有能力變得更好。」

📖 延伸思考

你是否曾經因為某位老師的一句話……

  • 從此變得更有自信，開始願意嘗試原本害怕的事？
  • 或是開始懷疑自己，覺得自己天生就不適合某件事？

很多人多年後忘了公式，
卻永遠記得：
當年老師曾怎麼看待自己。
